分布式事务

发表于 2021-06-23 分类于 分布式

在微服务环境下，服务与服务之间通讯采用 RPC 远程调用技术，但是每个服务中都有自己独立的数据源，即自己 独立的本地事务。两个服务相互通讯的时候，两个本地事务互不影响，从而出现分布式事务产生的原因。

在传统项目大部分情况下，不会产生分布式事务，但是在项目中如果采用多数据源方式，也会产生分布式事务。
