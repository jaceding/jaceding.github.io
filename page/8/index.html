Java集合-Map篇

发表于 2020-08-03 分类于 Java

JDK提供的集合类型主要分为四种类型：

  1. List：支持重复元素
  2. Set：不支持重复元素
  3. Map：键/值对的映射集
  4. Queue/Deque（double ended queue）：queue是在集合尾部添加元素，在头部删除元素的队列，deque是可在头部和尾部添加或者删除元素的双端队列，deque既可以实现队列又可以实现栈。

3种常用的缓存读写策略

发表于 2020-07-21 分类于 分布式

缓存的目的是为了减少数据库的压力，提升性能，但由于2个数据源之间是没有事务的，使用缓存可能会导致数据不一致的问题。

从理论上来说，给缓存设置过期时间，是保证最终一致性的解决方案。

常见的缓存读写策略有：

  • Cache Aside Pattern（旁路缓存模式）
  • Read/Write Through Pattern（读写穿透模式）
  • Write Behind Pattern（异步缓存写入模式）
